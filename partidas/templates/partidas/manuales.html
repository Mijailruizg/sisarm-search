Manual de Uso del Sistema SISARM Search

1. Inicio de sesión

  • Accede al sistema con tu nombre de usuario y contraseña.
  • Si no tienes cuenta, haz clic en “Crear nueva cuenta”.

2. Pantalla principal

  • Buscar Partidas
  • Historial
  • Manuales
  • Chat de Ayuda
  • Importar Excel (administradores)

3. Buscar partidas arancelarias

  • Haz clic en “Buscar Partidas”.
  • Escribe un código o palabra clave.
  • Presiona “Buscar” y revisa los resultados.
  • Haz clic en cualquier código para ver más detalles.

4. Ver historial

  • Accede a “Historial”.
  • Visualiza tus búsquedas anteriores con fecha.

5. Asistente virtual

  • Ingresa a “Chat de Ayuda”.
  • Escribe tu consulta (ej. “licencia”, “error”).

6. Licencia

Tu licencia debe estar activa para usar el sistema. Si aparece el mensaje “Licencia Expirada”, contacta al administrador.

7. Cerrar sesión

Puedes cerrar sesión desde la parte superior derecha.

Recomendaciones finales

  • Usa palabras clave claras para buscar.
  • Mantén tu licencia al día.
  • Consulta este manual si tienes dudas.
  • Contacta al administrador en caso de errores.
