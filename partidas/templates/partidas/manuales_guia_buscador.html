Guía rápida: Cómo usar el Buscador

Esta guía está pensada para usuarios que únicamente desean aprender a buscar aranceles de forma rápida y eficaz.

1. Búsqueda por código

  1. Escribe el código (total o parcial) en el campo de búsqueda. Ejemplo: 010121 o 0101.
  2. Pulsa Buscar o selecciona una sugerencia que aparece en el desplegable.
  3. Haz clic en el resultado para ver el detalle de la partida.

2. Búsqueda por descripción

  1. Escribe palabras clave que describan el producto (p. ej. "carne", "pescado", "medicamento").
  2. El sistema muestra partidas que contienen esas palabras en la descripción.
  3. Usa múltiples palabras para refinar resultados (por ejemplo: "carne fresca").

3. Autocompletado

Al escribir, el sistema muestra sugerencias basadas en código y descripción. Puedes seleccionar una sugerencia para completar el campo rápidamente.

4. Filtros

Abre el panel de filtros para limitar por:

  • Capítulo: familias de productos.
  • Gravamen: filtra por texto del impuesto.
  • Entidad que emite: Ministerio u organismo emisor.
  • Tipo de documento: si la partida requiere un permiso específico.

5. Resultado y detalle

En el resultado verás: Código, Descripción, Gravamen, ACE22 (CHI / PROT), y enlaces a referencias o documentos. Haz clic en el código para abrir la ficha detallada.

6. Qué hacer si no encuentras lo que buscas

  • Prueba variantes de palabras clave (singular/plural).
  • Revisa el Historial para ver búsquedas previas y ejemplos.
  • Usa el Chat de Ayuda para pedir orientación rápida: "Buscar partida" o "Ver manuales".
